# drag speak

## play speak text in the file

$ play_speak.bat [file path]

$ play_speak.bat hello.txt

or drag text file to play_speak.bat


## save speak to file 

$ save_speak [from file path] [to file path]

$ save_speak hello.txt hello.wav


## list voices

$ list_voices


## JavaScript Version

js_play_speak.bat

js_save_speak.bat
